El presente proyecto consta de una página web realizada en Python utilizando Django como framework.
La misma cuenta con una página principal de libre acceso que muestra la estructura básica de la web.

Toda la web cuenta con una barra de navegación semidinámica con herencia, que permite el retorno a la web de inicio, así como también el registro de nuevos usuarios (en caso de ya estar logueado este elementos es reemplazado por la edición del usuario), y el acceso a las distintas clases de datos que se decidieron crear.
Adicionalmente, en la esquina superior izquierda está el botón de login (o de logout en caso de ya estar logueado) y el avatar correspondiente al usuario. Cabe remarcar que en caso de no estar logueado, o tener un usuario sin avatar, el espacio en cuestión muestra un avatar genérico.
Es importante remarcar que tanto para visualizar los datos, como para editarlos o eliminarlos, es necesario estar logueado.

En las tres secciones de datos de la web (alumnos, docentes o cursos) la estructura es idéntica. En cualquiera de los casos se accede a un hub con 2 botones que cumplen la función de CRUD:

  • Listado:
    • Permite visualizar un listado de todos los elementos creados dentro de la clase correspondiente, ofreciendo las opciones de ver la información detallada, así como también editarla y eliminarla.

  • Formulario:
    • Permite ingresar un nuevo elemento a la clase en cuestión.
